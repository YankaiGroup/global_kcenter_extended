

# global_kcenter_extended

The implementation for the paper "Jiayang Ren, Ningning You, Kaixun Hua, Chaojie Ji, Yankai Cao (2025) A Global Optimization Algorithm for K-Center Clustering of One Billion Samples. Management Science."

Per journal policy, the source code is available as Supplemental Material at [https://doi.org/10.1287/mnsc.2023.00218](https://doi.org/10.1287/mnsc.2023.00218).
